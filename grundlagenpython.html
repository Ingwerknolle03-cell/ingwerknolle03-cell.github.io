1. Grundlagen

  • Kommentare: Mit # schreiben, z.B. # Das ist ein Kommentar
  • Einrückung: Python verwendet Einrückung statt geschweifter Klammern
  • Print: print("Hallo Welt") gibt Text aus
  • Python Version prüfen: python --version

2. Variablen & Datentypen

  • Integer: Ganze Zahlen, z.B. x = 5
  • Float: Dezimalzahlen, z.B. pi = 3.14
  • String: Text in Anführungszeichen, z.B. name = "Anna"
  • Boolean: Wahr/Falsch, z.B. is_active = True

3. Operatoren

  • Arithmetische Operatoren: +, -, *, /, %, **, //
  • Vergleichsoperatoren: ==, !=, >, <, >=, <=
  • Logische Operatoren: and, or, not
  • Zuweisung: =, +=, -=, *=, /=

4. Bedingungen (if)

  • Syntax: if Bedingung:
  • Optional: elif und else
  • Beispiel: 
    x = 10
if x > 5:
    print("Groß")
else:
    print("Klein")

5. Schleifen (for & while)

  • for-Schleife: Iteration über eine Liste oder Range 
    for i in range(5):
    print(i)
  • while-Schleife: Wiederholt solange Bedingung True 
    x = 0
while x < 5:
    print(x)
    x += 1

6. Funktionen

  • Definition: def meine_funktion():
  • Beispiel: 
    def add(a, b):
    return a + b

print(add(3,4))  # 7

7. Listen & Dictionaries

  • Liste: numbers = [1,2,3]
  • Zugriff: numbers[0]
  • Dictionay: person = {"name":"Anna", "age":25}
  • Zugriff: person["name"]

8. Strings

  • Verkettung: "Hallo " + "Welt"
  • Länge: len("Hallo")
  • Indexing: s[0] für erstes Zeichen
  • Formatierung: f"Name: {name}"

9. Module & Bibliotheken

  • Import: import math
  • Benutzen: math.sqrt(16)
  • Eigene Module erstellen mit .py Dateien

10. Tipps & Best Practices

  • Kommentare für Übersichtlichkeit nutzen
  • Variablen sinnvoll benennen
  • Einrückung korrekt verwenden
  • Regelmäßig testen und debuggen
  • Dokumentation und Online-Ressourcen lesen
