1. Grundlagen
  • Kommentare: # Kommentar
  • Einrückung ist Pflicht
  • Ausgabe: print("Hallo Welt")
2. Variablen & Datentypen
  • Integer: x = 5
  • Float: pi = 3.14
  • String: name = "Anna"
  • Boolean: is_active = True
3. Operatoren
  • + - * / % ** //
  • == != > < >= <=
  • and or not
4. Bedingungen
x = 10
if x > 5:
    print("Groß")
else:
    print("Klein")
5. Schleifen
for i in range(5):
    print(i)

x = 0
while x < 5:
    print(x)
    x += 1
6. Funktionen
def add(a, b):
    return a + b

print(add(3, 4))  # 7
7. Listen & Dictionaries
numbers = [1, 2, 3]
print(numbers[0])   # 1

person = {"name": "Anna", "age": 25}
print(person["name"])  # Anna
8. Strings
"Hallo " + "Welt"    # Hallo Welt
len("Hallo")          # 5
s[0]                  # erster Buchstabe
f"Name: {name}"       # f-String
9. Module
import math
math.sqrt(16)   # 4.0
10. Tipps
  • Kommentare verwenden
  • Variablen sinnvoll (auf Englisch) benennen
  • Einrückung korrekt setzen
  • Eine bestimmte Fachrichtung aussuchen
  • Quiz machen und selbst ausprobieren
